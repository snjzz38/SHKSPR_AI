API Key Setup Tutorial

The "Limit Exceeded" error is due to free plan limitations.
However, you can use your own free API key to bypass these issues.

Step 1: Obtain your API Key

The AI Humanizer uses the Groq API Key. ALL OTHER APPLICATIONS use the Gemini API Key.

Groq API Key:

Go to https://console.groq.com/keys, create a free account, then click "Create API Key". Give it any name and copy the value it gives you.

Gemini API Key:

Go to https://aistudio.google.com/u/1/apikey?pli=1, sign in, accept any Terms of Service, then click "Create API Key". Copy the value it gives you.

IMPORTANT: Save your API key! Do not share it publicly.

Step 2: Use the API Key in the Application

On any of the AI tools, proceed to the final step before generation. There you will find an "Advanced Options" button.

1. Click the "Advanced Options" button to reveal the API key input field.

2. Paste your copied API key into the input field that appears.

3. Click "Set Key for Session". You will see a confirmation message. Your key is now active for this session only.

Step 3: Enjoy!

You can now use the application without rate limits. The key is not saved, so you will need to re-enter it if you close or refresh the page. You will likely never need to create a new API key from Google or Groq.
