API Key Local Setup Tutorial

The "Limit Exceeded" error is due to the free plan limitations I have. However, you can run the application yourself to avoid these issues.

Step 1: Obtain your API Key

The AI Humanizer uses the Groq API Key
ALL OTHER APPLICATIONS use the Gemini API Key

Groq API Key:

Go to https://console.groq.com/keys and make an account (its free!), then click "Create API Key", give it a name (whatever you want). And Copy the value it gives you.

Gemini API Key:

Go to https://aistudio.google.com/u/1/apikey?pli=1, sign in if you need to, and accept whatever Terms of Service they give you. Then click "Create API Key". Copy the value it gives you.

IMPORTANT: Save your API key, you will need it later! Don't share it publicly as it's a security threat.

Step 2: Add the API Key into the code:

Go to https://github.com/snjzz38/SHKSPR_AI/tree/main/Local_Use, pick a project to run like Humanizer.html or Grader.html for example. Copy the file text.

Go to https://www.codechef.com/html-online-compiler, and go paste the code under the "index.html" section (IMPORTANT!), then click in the text and do ctrl f and search for "your-api-key-here", replace the "your-api-key-here" with the API key you made before, run the code and enjoy your working code!

Step 3: Enjoy!

You can save this edited version of the code to use whenever you want!

Keep in mind that the code may update, you can check the Github Folder https://github.com/snjzz38/SHKSPR_AI/tree/main/Local_Use for any updates.
You would probably NEVER need to make a new API key as the code updates separately.
