API Key Local Setup Tutorial

The "Limit Exceeded" error is due to free plan limitations.
However, you can run the application yourself to avoid these issues.

Step 1: Obtain your API Key

The AI Humanizer uses the Groq API Key. ALL OTHER APPLICATIONS use the Gemini API Key.

Groq API Key:

Go to https://console.groq.com/keys, create a free account, then click "Create API Key". Give it any name and copy the value it gives you.

Gemini API Key:

Go to https://aistudio.google.com/u/1/apikey?pli=1, sign in, accept any Terms of Service, then click "Create API Key". Copy the value it gives you.

IMPORTANT: Save your API key! Do not share it publicly.

Step 2: Add the API Key into the code:

Go to the project's Local Use Folder on GitHub and select a tool like `Humanizer.html`. Copy the entire file text.

Go to an online editor like CodeChef's HTML Compiler, paste the code into the "index.html" section. Press Ctrl+F, search for "your-api-key-here", and replace it with the API key you copied earlier. Run the code and enjoy!

Step 3: Enjoy!

You can save this edited code to use whenever you want. Keep in mind that the code may update, so you can check the GitHub folder for new versions. You will likely never need to create a new API key.
